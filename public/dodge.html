Dodge

voir sur Play Store

Dodge est un jeu d'arcade inspiré des années 80 disponible sur Play store !

Le but : survivre le plus longtemps possible.

À l'origine, Dodge est mon projet d'ISN en Terminale. Développé de février à mai 2019, il était initialement codé avec Processing (java) pour pc. Durant l'été, j'ai refait tout le code en "vrai" Java avec Android Studio. La version mobile est sortie en novembre 2019. C'est avec ce jeu que l'histoire de Bedjes Studio a commencé.

De nombreuses fonctionnalités ont été ajoutés à la version de base : ajout de nouveaux avions, un nouveau mode de jeu, la création d'un système de mission ou encore les achats intégrés.

Aujourd'hui, le jeu à une note de 4,9/5 et la barre des 1500 téléchargements a été franchie !
